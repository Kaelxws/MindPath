MindPath – Website Institucional (Projeto Front-End)

Este é um projeto fictício desenvolvido para treinar e demonstrar habilidades em HTML, CSS e JavaScript básico.
A MindPath é uma empresa britânica especializada em consultoria de saúde mental cognitiva, focada em ajudar equipes a melhorarem foco, equilíbrio e produtividade no ambiente de trabalho.



 Objetivo do Projeto

Criar um site institucional moderno, minimalista e responsivo que apresente:

A identidade visual da empresa

Serviços oferecidos

Depoimentos de clientes

Chamada final para contato

Um layout profissional digno de portfólio



Tecnologias Utilizadas

HTML5 – estruturação das páginas

CSS3 – estilização e responsividade

JavaScript básico – pequenas interações (scroll, menu, etc.)

Figma / IA – para criação da logo (PNG transparente)
